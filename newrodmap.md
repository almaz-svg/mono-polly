# Что делать дальше с MarketWars

## Шаг 1 — Задеплоить на Vercel (сегодня)

1. Зайти на vercel.com → New Project → выбрать mono-polly
2. Добавить 4 переменные окружения (смотри DEPLOY.md)
3. Нажать Deploy
4. Проверить что сайт открывается и `/admin` работает

---

## Шаг 2 — Получить Gemini ключ (сегодня)

1. Зайти на aistudio.google.com/apikey
2. Создать ключ — он начинается с `AIzaSy...`
3. Вставить в Vercel → Settings → Environment Variables → Redeploy
4. Проверить AI-оценку в `/admin`

---

## Шаг 3 — Протестировать игру (до хакатона)

1. Открыть `/admin` → создать игру
2. Открыть `/register` с другого устройства → зарегистрировать 2-3 команды
3. Начать игру → команды заходят на `/team/:id`
4. Команды сдают фичи → запустить AI-оценку
5. Проверить что `/dashboard` обновляется в реальном времени

---

## На хакатоне

1. Открыть `/dashboard` на проекторе
2. Каждая команда заходит на сайт с телефона → `/register`
3. Ты управляешь игрой с `/admin`
4. После каждого раунда: завершить раунд → AI-оценка → расчёт акций → событие → следующий раунд
