## Шаг 1 — Получить рабочий Gemini ключ

Текущий ключ не работает. Нужен новый:

1. Зайти на aistudio.google.com/apikey
2. Нажать **Create API key** — ключ начинается с `AIzaSy...`
3. В Vercel → твой проект → **Settings → Environment Variables**
4. Обновить `VITE_GEMINI_API_KEY` → нажать **Redeploy**

---

## Шаг 2 — Протестировать полный цикл игры

1. `/admin` → войти → создать игру
2. `/register` с другого устройства → зарегистрировать 2-3 команды
3. `/admin` → начать игру
4. Команды заходят на `/team/:id` → вводят пароль → сдают фичи
5. `/admin` → завершить раунд → AI-оценка → расчёт акций → событие
6. `/dashboard` → проверить что график обновляется в реальном времени

---

## Шаг 3 — На хакатоне

1. Открыть `/dashboard` на проекторе
2. Команды заходят с телефонов на `/register`
3. Ты управляешь с `/admin`
4. Повторять раунды до конца игры
